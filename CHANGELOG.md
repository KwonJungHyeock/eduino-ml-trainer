# 개발 기록 (CHANGELOG)

Eduino AI Lab — 브라우저에서 직접 배우는 AI 학습·추론 + 영상처리 체험 도구.
교사·학생 상호작용을 목표로, 카메라·마이크 데이터를 브라우저 안에서만 처리합니다.

---

## v1.0.0

### 패키징 / 배포
- **Electron 데스크톱 앱**(`electron-main.js`) 추가 — 바탕화면에서 더블클릭으로 실행되는 독립 앱(포터블 .exe). 권한 핸들러로 카메라/마이크 허용.
- `npm run app`(개발 실행) / `npm run dist`(포터블 .exe 빌드) 스크립트.

### 기능 구성 (좌측 사이드바 · config 기반)
- **AI 학습·추론(만드는 AI)**: 이미지 분류(MobileNet) · 음성 분류(Speech Commands) · 동작 분류(MoveNet) · 텍스트 분류(준비중)
- **AI 영상처리(체험하는 AI)**: 영상 효과/필터 · 색상 추적 · 얼굴 검출(BlazeFace) · 뼈대 검출(MoveNet) · 객체 탐지(COCO-SSD)
- 기능 메타데이터(그룹·아이콘·뱃지·단계·이론)를 단일 `FEATURES`/`NAV_GROUPS`/`THEORY` config 로 분리 → 기능 추가 = 데이터 추가.

### 주요 학습/체험 흐름
- 클래스 정의 → 데이터 수집(단발/연속·녹음) → 학습(학습곡선·혼동행렬) → 실시간 추론(신뢰도 막대 + **도넛 게이지**).
- **StepNav**: 학습 진행(클래스/데이터/학습/추론)을 단계 칩에 done·active 로 실시간 표시.
- 이론 영역: 기능별 **학습 정보 / 알고리즘 정보**(K-12 눈높이).
- 음성 녹음 관리(스펙트로그램 썸네일 + 개별 삭제), 이미지 샘플 갤러리.
- 영상처리 파라미터: 밝기/대비/블러(필터), 색 허용 범위·직접 색 입력(색상 추적), 관절 민감도(뼈대), 신뢰도 임계값(객체 탐지).

### 디자인
- 다크 테마 + 코랄 포인트(Eduino 톤), 사이드바 영역 구분·대문 헤더, 정사각 카메라, 게이지/도넛 시각화.

### 품질
- jsdom 스모크/회귀 테스트 23종(브라우저 없이 로드·학습·추론·UI 검증).

## 개발 과정 요약 (이 버전까지의 여정)

### 1. 구조 점검 · 정본화 · 안정화
- 분산돼 있던 앱 본체를 `index.html` 로 일원화.
- **탭 전환 시 카메라/마이크/추론 루프 미정리** 버그 수정(`stopAllActivity`).
- jsdom 테스트 하니스 도입(tf/모델 스텁) → 브라우저 없이 핵심 경로 검증.
- 중복 렌더/학습/혼동행렬 로직 공통 헬퍼로 추출. **혼동행렬 use-after-dispose** 버그를 테스트로 잡아 수정.
- 사전학습 탭 실사용 버그 정리: 객체탐지 박스 좌우반전, 인물분리 검정화면(비디오 직접 노출 + 투명 오버레이), 깊이추정 제거 등.

### 2. 정보 구조 개편
- 상단 3탭 → **좌측 사이드바 2영역**(만드는 AI / 체험하는 AI), config 기반 생성.
- **StepNav**(번호 단계 칩), 기능 정보 카드 + 이론 영역, 신규 기능(얼굴·뼈대 검출), 알고리즘별 파라미터 슬라이더, 추론 도넛 게이지.

### 3. 디자인 / 마감
- 전체 **다크 테마** 전환 + 코랄 브랜딩, 로고/배너 정비, 폰트 가독성(한글 mono 폴백) 개선.
- 영상 효과 바둑판 토글, 색상 추적 프리셋 확장 + 직접 색 입력, 클래스 네이밍 통일.
- 불필요 상단 뱃지 제거, INFO 패널 밸런스.

상세 변경은 git 커밋 로그 참고.
